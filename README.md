# RT-StackChan コントローラー

RT-StackChanロボットをWebブラウザからソケット通信で遠隔操作するためのコントローラーアプリです。

## 動作環境

- Python 3.x
- [uv](https://github.com/astral-sh/uv) (依存関係管理に使用)

## セットアップ

本プロジェクトは `uv` を使用して初期化されています。

```bash
uv sync
```

## アプリケーションの実行方法

ロボット（またはモックサーバー）と、Webコントローラーの2つのプロセスを実行する必要があります。

### 1. ロボットモックサーバーの起動 (ターミナル1)
ロボット実機がない場合の動作検証用サーバーです。コマンドを受信してログ表示します。

```bash
uv run python robot_mock.py
```

### 2. Webコントローラーの起動 (ターミナル2)
FastAPIサーバーを起動します。

```bash
uv run uvicorn app.main:app --reload
```

## 使い方

1.  ブラウザで [http://127.0.0.1:8000](http://127.0.0.1:8000) にアクセスします。
2.  **Head Control**: 四角い領域をドラッグして、ロボットの首振り（Pan/Tilt）を制御します。
3.  **Expressions**: ボタンをクリックして、ロボットの表情を変更します。
4.  **Speech**: テキストを入力してSendボタンを押すと、ロボットが発話します。

## 構成設定

- **接続先**: 現在は `app/main.py` 内にハードコードされています (`127.0.0.1:9999`)。実機と接続する場合はここを変更してください。
